## 기술 스택 (정확한 명칭 사용)

- **AI 엔진**: Groq API — LLaMA 3.3 70B Versatile 모델
- **이미지 생성**: Pollinations AI (flux 모델) → LoremFlickr (대체)
- **웹 인터페이스**: Streamlit (Python)
- **메신저 채널**: 카카오톡 (KakaoTalk) 웹훅, 텔레그램 (Telegram) 봇
- **블로그 플랫폼**: 구글 블로거 (Google Blogger)
- **이미지 형식**: Base64 인코딩 (Data URI) — 복붙 호환성 보장

## 콘텐츠 스타일

- **시점**: 반드시 1인칭 (나는, 내가, 저는)
- **언어**: 순수 한국어(한글)만. 한자·영어 단어 혼용 금지
- **구조**: 서론(Chapter 1) — 본론(Chapter 2) — 결론(Chapter 3)
- **챕터당 분량**: 최소 5문장 이상
- **감성 방향**: 따뜻함, 회고, 일상의 소중함, 감사함
- **디자인 테마**: Liquid Glass Aesthetic (유리 질감, 다크 배경)

## 배리어 프리 (Barrier-Free) 원칙

- 새로운 앱 설치 없이 카카오톡으로만 사용 가능
- 메모 한 줄만 보내면 AI가 나머지 전부 처리
- 결과는 링크 1개로 공유 (복잡한 조작 불필요)
- 어르신·청소년 모두 사용 가능한 초간단 UI
